 # Customer Churn

This repository represents an example of comparing Machine Learning models using the [Kaggle Customer Churn database](https://www.kaggle.com/datasets/muhammadshahidazeem/customer-churn-dataset).

The models tested were

* Logistic Regression
* Support Vector Machine
* Random Forest
* XG Boost

Various hyperparameters were tested for some of the models.

The best model was XG Boost.
